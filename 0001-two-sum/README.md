## Problem: Two Sum
## Difficulty: Easy
## Platform: LeetCode

## Approaches

### 1. Brute Force
- Time: O(n²)
- Space: O(1)

### 2. HashMap
- Time: O(n)
- Space: O(n)

### 3. Optimized
- Same as HashMap but cleaner implementation


Why I used this approach:
- brute force first step to understand problem
- hashmap reduces lookup time to O(1)
